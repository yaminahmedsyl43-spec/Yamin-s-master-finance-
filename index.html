Deploy your google-apps-script.gs file as a Web App and paste the URL here

🔒 Setup Guide

  1. Open your Google Sheet → click Extensions → Apps Script
  2. Delete all existing code, paste in the contents of google-apps-script.gs
  3. Click Deploy → New deployment → Web App
  4. Set "Execute as" = Me, "Who has access" = Anyone
  5. Click Deploy → Authorise → copy the Web App URL
  6. Paste it above and click Save Settings
